## 📌 프로젝트 소개 (Project Overview)
**이름뭐로할까**은 세종대학교 학생들의 안전과 편의를 위해 개발된 **AI 기반 시설물 고장 신고 앱**입니다. 
기존의 복잡한 전화 중심 신고 방식에서 벗어나, 고장 부위의 사진을 찍기만 하면 AI가 즉각적으로 문제 유형을 분석하고 위치와 함께 관리자에게 전달합니다.

* **Target:** 세종대학교 재학생 및 교직원
* **Core Value:** 신속성(Quick), 정확성(Accurate), 편의성(Easy)

## ✨ 주요 기능 (Key Features)
- 📸 **AI 사진 분석**: 사진 촬영 시 AI(YOLO/GPT-4o Vision)가 고장 유형(파손, 전등 꺼짐, 수전 누수 등)을 자동 판별합니다.
- 📍 **위치 기반 접수**: 교내 지도를 활용하여 사용자가 고장 발생 위치를 특정합니다.
- 📊 **실시간 진행 상태**: 신고한 시설물의 수리 진행 단계(접수/수리중/완료)를 바로바로 확인합니다.
- 📱 **User-Friendly UI**: React Native Paper를 활용한 세종대학교 아이덴티티가 담긴 깔끔한 디자인.
